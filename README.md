# Arduino Radar Using Ultrasonic Sensor

This project detects obstacles using an ultrasonic sensor connected to Arduino and displays object position in radar-style interface.

This project was developed as part of a team academic project. My contribution included sensor interfacing, circuit setup, and testing.

Components Used:
- Arduino UNO
- Ultrasonic Sensor (HC-SR04)
- Servo Motor

Working Principle:
The ultrasonic sensor sends sound waves and receives reflected signals from objects. Distance is calculated based on echo time and displayed on radar-style interface.

Applications:
- Robotics navigation
- Obstacle detection systems
- Smart parking assistance
